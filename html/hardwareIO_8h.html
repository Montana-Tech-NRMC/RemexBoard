init_switches will initialize hardware interrupts to read limit switches or buttons.

Parameters
[in]switch_ais the callback function that will be invoked when the limit switch connected to pin 2.4 on the micro-controller is triggered.
[in]switch_bis the callback function that will be invoked when the limit switch connected to pin 2.5 on the micro-controller is triggered.

if either parameter is null do not initialize hardware for the associated pin.
